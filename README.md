# Simple Chat AI

A minimal React + Vite chat interface that sends messages to a webhook and displays the response.

## Run locally

```sh
npm install
npm run dev
```

The app runs on `http://localhost:8080` by default.

## Webhook configuration

Current webhook URL (kept as-is):

`http://localhost:5678/webhook/chatapp`

You can find and change it in:

`src/pages/Index.tsx`

## Tech stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
